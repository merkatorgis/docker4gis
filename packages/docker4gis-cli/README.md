# docker4gis CLI tool

## Install

```
npm install -g docker4gis
```

## Initialise new project

```
dg init
```

`dg new` works as well.

## Add component to project

```
dg component
```

`dg generate component` or `dg g c` work as well.


## Or initialise in Azure DevOps

```
dg devops
```

## Help

List commands:

```
dg
```

Read instructions:

```
dg help [COMMAND]
```
